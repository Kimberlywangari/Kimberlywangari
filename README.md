## 🚀 Featured Projects

### 🌿 [TwinFarm AIware](https://github.com/loskiii/twinfarm-digital-twin) · [Live Demo](https://twinfarm.netlify.app/)
A **digital-twin platform** that mirrors a coriander field in software, fusing live IoT sensor data with **explainable ML** to give smallholder farmers early crop-stress warnings, growth-stage detection, and yield projections. I led the **data science & ML**: three models (CatBoost classifiers + a deterministic yield estimator) with **SHAP** explainability and a domain-adaptation pipeline bridging Kaggle data and Raspberry Pi field sensors.
**Stack:** Python · CatBoost · SHAP · scikit-learn · Flask · FIWARE · Three.js

### 🏃 [Kimbia Eldoret](https://github.com/Kimberlywangari/kimbia-eldoret)
A **production website** built for a real running club in Eldoret — mobile-first, with animated stat counters, an interactive FAQ, an infinite-loop community carousel, and live Google Calendar, Strava, and WhatsApp integrations. Shipped end-to-end for an actual community organization.
**Stack:** HTML5 · CSS3 · JavaScript (Intersection Observer API)
